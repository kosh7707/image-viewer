# ImageViewer

Windows 이미지 뷰어

지원: `.jpg`, `.jpeg`, `.png`, `.webp`, `.gif`

## 다운로드

Releases에서 `ImageViewerPortable-*-win-x64.zip` 다운로드:

압축 풀고 실행:

```text
ImageViewerPortable.cmd
```

우클릭 메뉴 연동(컨텍스트 메뉴)은 Settings에서 켜고 끔

## 조작

| 키    | 동작                         |
| ----- | ---------------------------- |
| `←`   | 이전                         |
| `→`   | 다음                         |
| `F`   | 전체화면                     |
| `[`   | GIF/WebP 느리게              |
| `]`   | GIF/WebP 빠르게              |
| `Esc` | 종료. 단, 대화상자 없을 때만 |

우클릭: 파일 열기 / 폴더 열기 / 정렬 / 설정 / 종료
